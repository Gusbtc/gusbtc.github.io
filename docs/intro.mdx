# BebopC2

**BebopC2** is a modular, asynchronous command and control (C2) framework engineered for authorized security assessments and red team engagements.

Designed with an OPSEC-first philosophy, BebopC2 minimizes the forensic footprint by moving high-risk capabilities like .NET execution into a minimalist, BOF-centric architecture.

:::warning Authorized Operation Only
BebopC2 is a professional security tool. As an operator, you are responsible for maintaining explicit authorization and adhering to the defined scope of your engagement.
:::

## The Operator's Philosophy

BebopC2 is built on the principle that the framework should be as quiet as the operator.

*   **Avoid Process Creation:** Utilize in-process execution for situational awareness tasks.
*   **Minimize Disk Artifacts:** Prioritize memory-resident modules over traditional file-based staging.
*   **Context-Aware Execution:** Choose specialized commands and modes based on the target's environment.


## Core Architecture

The core agent is designed with minimal static capability bloat, moving advanced features into on-demand, reflectively loaded modules.
